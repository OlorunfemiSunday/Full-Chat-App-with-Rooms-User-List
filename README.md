Full Real-Time Chat App (Socket.io Basics)
📌 Project Description

This is a full real-time chat application built using Node.js, Express, and Socket.io.
It allows multiple users to join, send, and receive messages instantly in real time.

⚙️ Tech Stack Used

Node.js – Backend runtime

Express – Web framework

Socket.io – Real-time bidirectional communication

🚀 Setup Instructions

Clone the repository

git clone https://github.com/<your-username>/Full-Chat-App-with-Rooms-User-List.git
cd Full-Chat-App-with-Rooms-User-List


Install dependencies

npm install


Run the server

node server.js


Open in browser
Go to: http://localhost:3000

(Open in two or more tabs to test the chat in real time)

📖 Usage Examples

Open two browser tabs at http://localhost:3000.

Type a message in one tab and see it instantly appear in the other.
